How it was build

The site is built by Hugo with Hugo-Octopress theme.

I have bumped into this theme coinsidentaly on this blog http://blog.pnkfx.org/.

If you guess what is the thing in the header. This is my take on awesome octupus from the original theme http://octopress.org/ page. The idea that it resembles a shot from Ninja Turtles tv series.

We override the original theme a little bit to be more compatible with original site. We do it by creating our own layouts. The approach is inspired by https://github.com/parsiya/parsiya.net.
