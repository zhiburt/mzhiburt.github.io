Cope with Garmin && Strava limits

One upon a time there has been a pile of activitis in Garmin Connect which was not synced with Strava. It was thought it will be an easy thing to do to just syncronize them via some import/export feature. But it turned out not to be the case.

A case agains automatic syncranization

You can sync your Garmin Connect and Strava directly? But in such case you'll import/export all data available.

So you'll open your pace,hr etc on Strava which is sometimes too much.

But we could deal with this via manual import/export. This way you're in charge (to a degree).

To be in controll of your data you might better not to use such services at all. You might not better use Internet too.

File syncranization

  • Garmin only supports an export of all activities in CSV.
  • At the same time Strava doesn't support CSV import.
  • Strava can import GPX files.
  • Which Garmin can export only 1 by 1 activity. Which is to be least to say quite inefective.

Downloading all activities

The javascript script below can be used to download all activities 1 by 1 in GPX format.

To do so put it in your browser console.

jQuery.getJSON(
    'https://connect.garmin.com/modern/proxy/activitylist-service/activities/search/activities?limit=200',
    function(act_list)
    {
        var t=0;
        act_list.forEach(
        function(act)
            {
                setTimeout(function() {
                    console.dir(act['activityId'], act['activityName'], act['startTimeLocal']);
                    location = 'https://connect.garmin.com/modern/proxy/download-service/export/gpx/activity/' + act['activityId'];
                }, t+=5000);
            }
        );
    }
);

Strava upload

Then you can use https://www.strava.com/upload/select to import the activities.

Though be wary that there's a limit on ~25 activities per day you can sync this way.

Removing HR data

Usage

python remove_extensions.py C:\Users\xxxx\Downloads\garmin_data

Script

The python script bellow can be used to do so.

import os
import os.path
import sys
import xml.etree.ElementTree as ET

def removeExtensions(xmlstr):
    ET.register_namespace('', "http://www.topografix.com/GPX/1/1")
    ET.register_namespace('xsi', "http://www.w3.org/2001/XMLSchema-instance")

    root = ET.fromstring(xmlstr)
    trkseg_lst = root.findall('./{http://www.topografix.com/GPX/1/1}trk/{http://www.topografix.com/GPX/1/1}trkseg/')
    for entry in trkseg_lst:
        for element in list(entry):
            if element.tag == "{http://www.topografix.com/GPX/1/1}extensions":
                entry.remove(element)

    return ET.tostring(root)

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Expected to get a path to an activity folder as a first argument")
        exit(-1)

    activityFolder = sys.argv[1]

    print("Opening the provided folder: ", activityFolder)

    for filename in os.listdir(activityFolder):
        filePath = os.path.join(activityFolder, filename)
        if os.path.isdir(filePath):
            continue

        print("Converting file: ", filename)

        converted = ""
        with open(filePath, 'r') as f:
            data = f.read()
            converted = removeExtensions(data)

        with open(filePath, 'wb') as f:
            f.write(converted)

Conclusion

  2. We can edit data.
  3. We can upload it.
